Quick intro to REST

What is REST?
REST is a set of conventions for creating APIs which operate over HTTP. To put it in simpler terms, REST offers some guidelines on how an application and server should interact over the web.

What does REST stand for?
Good question. REST stands for REpresentational State Transfer. An API is said to be RESTful, if it follows the REST guidelines.
